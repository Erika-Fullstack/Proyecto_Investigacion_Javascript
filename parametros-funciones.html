Parámetros en funciones JavaScript

1: Parámetros básicos (marcadores de posición)
        
                      function saludar(nombre) {
                        console.log(`¡Hola, ${nombre}!`);
                    }
                    
                    saludar("Marta"); // Salida: ¡Hola, Marta!
                    saludar("Carlos"); // Salida: ¡Hola, Carlos!

nombre es un parámetro que actúa como un marcador de posición. Se sustituye por el valor que le pases al llamar a la función.

2: Parámetros predeterminados
        
                      function saludar(nombre = "Amigo") {
                        console.log(`¡Hola, ${nombre}!`);
                    }

                    saludar();           // Salida: ¡Hola, Amigo!
                    saludar("Sofía");    // Salida: ¡Hola, Sofía!

El valor "Amigo" es el valor predeterminado del parámetro nombre. Si no se pasa ningún argumento, se utiliza el valor por defecto.

3: Parámetros rest
        
                      function listarFrutas(...frutas) {
                        console.log("Frutas recibidas:");
                        frutas.forEach((fruta) => console.log(fruta));
                      }
              
                      listarFrutas("Manzana", "Plátano", "Cereza");

Al definir el parámetro (...frutas) le decimos a la función que todos los parámetros que reciba los convierta en un array. Esto nos es útil cuando no sabemos el número de argumentos que va a recibir una función en concreto

4: Valores de retorno
        
                      function obtenerAreaRectangulo(base, altura) {
                        return base * altura; // Devuelve el área
                      }
                    
                      let area = obtenerAreaRectangulo(5, 3);
                      console.log(`El área del rectángulo es ${area}`); 
                      // Salida: El área del rectángulo es 15

La función obtenerAreaRectangulo devuelve un valor (el producto de base y altura). Este valor puede almacenarse en una variable o utilizarse directamente.

5: Funciones flecha con parámetros
        
                      // Manera clásica de escribir una función
                      const multiplicar = function (a, b) {
                        return a * b;
                      };
                      
                      // Manera moderna de función flecha
                      const multiplicar = (a, b) => a * b;

                      console.log(multiplicar(4, 5)); // Salida: 20

Las funciones flecha ofrecen una sintaxis más compacta y no definen su propio this.

6: Errores al acceder a parámetros fuera de la función
        
                        function mostrarMensaje(mensaje) {
                          console.log(mensaje);
                      }
                      
                      mostrarMensaje("¡Hola!"); 
                      // Salida: ¡Hola!
                      
                      // console.log(mensaje); 
                      // Error: mensaje no está definido fuera de la función

El parámetro mensaje solo existe dentro del alcance (scope) de la función mostrarMensaje.
