Parámetros en funciones JavaScript

Ejemplo 1: Parámetros básicos (marcadores de posición)
        
    function saludar(nombre) {
      console.log(`¡Hola, ${nombre}!`);
  }
  
  saludar("María"); // Salida: ¡Hola, María!
  saludar("Carlos"); // Salida: ¡Hola, Carlos!

nombre es un parámetro que actúa como un marcador de posición. Se sustituye por el valor que le pases al llamar a la función.

Ejemplo 2: Parámetros predeterminados
        
    function saludar(nombre = "Amigo") {
      console.log(`¡Hola, ${nombre}!`);
  }

  saludar();           // Salida: ¡Hola, Amigo!
  saludar("Sofía");    // Salida: ¡Hola, Sofía!

El valor "Amigo" es el valor predeterminado del parámetro nombre. Si no se pasa ningún argumento, se utiliza el valor por defecto.

Ejemplo 3: Parámetros rest
        
    function sumar(...numeros) {
      return numeros.reduce((acumulador, numero) => acumulador + numero, 0);
  }
  
  console.log(sumar(1, 2, 3));             // Salida: 6
  console.log(sumar(4, 5, 6, 7, 8));      // Salida: 30
  console.log(sumar());                   // Salida: 0
Ejemplo 4: Valores de retorno
        
    function obtenerAreaRectangulo(base, altura) {
      return base * altura; // Devuelve el área
    }
  
    let area = obtenerAreaRectangulo(5, 3);
    console.log(`El área del rectángulo es ${area}`); // Salida: El área del rectángulo es 15

La función obtenerAreaRectangulo devuelve un valor (el producto de base y altura). Este valor puede almacenarse en una variable o utilizarse directamente.

Ejemplo 5: Funciones flecha con parámetros
        
      const calculadora = {
        sumar: function(a, b) {
            return a + b;
        },
        restar: function(a, b) {
            return a - b;
        }
      };

      console.log(calculadora.sumar(3, 5)); // Salida: 8
      console.log(calculadora.restar(9, 4)); // Salida: 5

Las funciones flecha ofrecen una sintaxis más compacta y no definen su propio this.

Ejemplo 6: Función flecha con rest y retorno implícito
        
  const sumarNumeros = (...numeros) => numeros.reduce((acum, num) => acum + num, 0);

  console.log(sumarNumeros(10, 20, 30)); // Salida: 60

Combina el operador rest con una función flecha. No es necesario usar return porque el retorno es implícito.

Ejemplo 7: Errores al acceder a parámetros fuera de la función
        
    function mostrarMensaje(mensaje) {
      console.log(mensaje);
  }
  
  mostrarMensaje("¡Hola!"); // Salida: ¡Hola!
  
  // console.log(mensaje); // Error: mensaje no está definido fuera de la función

El parámetro mensaje solo existe dentro del alcance (scope) de la función mostrarMensaje.
