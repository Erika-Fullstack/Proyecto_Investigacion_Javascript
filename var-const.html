Variables y Constantes

Ejemplo Símbolo (Symbol): Identificador único utilizado para propiedades de ojetos. 
                    
    // Declaramos una variable Symbol 
    let uniqueId = symbol ("id")
    let hiddenKey = Symbol("hidden")
            
              
    // Llamamos a la variable
    console.log(uniqueId); //salida: Symbol(id)
    console.log(hiddenkey); //salida: Symbol(hidden)
Ejemplo Nulo (Null): Indica que una variable se ha declarado pero no se le ha asignado un valor. 
                    
    // Declaramos una variable NUll 
    let selectedProduct = null;
            
              
    // Llamamos a la variable
    console.log(selectedProduct); // salida: null
Ejemplo Indefinido (Undefined): Cuando una variable se ha declarado pero no se le ha asignado un valor.

Ejemplo Funciones (Function): Un bloque de código reutilizable que puede ser llamado. 
                    
    // Declaramos una variable Function
    let greet = function(name) {
    return `Hola, ${name}!`;
    };
            
              
    // Llamamos a la variable
    console.log(greet("Carla")); //salida: "Hola,Carla!"

Ejemplo Set: Almacena valores únicos, ideal para eliminar duplicados o verificar la existencia de elementos. 
                
    // Creamos un Set
    let mySet = new Set ();

    //Agregar algunos valores al Set
    mySet.add (1)
    mySet.add (2)
    mySet.add (3)

          
    // Llamamos a la variable 
    console.log (mySet); //salida: Set { 1, 2, 3}

Ejemplo RegExp:Útil para realizar búsquedas y validaciones en texto mediante patrones.

    let regex= "hello;"

    //Usamos la expresión regular 
    let text= "Hello, world!";

    // BUscamos coincidencias
    let result = text.match (regex)

          
    // Llamamos a la variable
