Variables y Constantes

Ejemplo Número: Variable Número (Number): Almacena valores númericos (enteros y decimales)
Ejemplo Cadena (String): Contiene secuencias de caracteres, como texto.
Ejemplo Booleano (Boolean): Representa valores verdaderos (true) o falsos (false).
Ejemplo Nulo (Null): Indica que una variable se ha declarado pero no se le ha asignado un valor.
Ejemplo Indefinido (Undefined): Cuando una variable se ha declarado pero no se le ha asignado un valor.
Ejemplo Arreglo (Array):Almacena una lista ordenada de elementos.
Ejemplo Objeto (Object): Estructura de datos que puede contener propiedades y métodos.
Ejemplo Funciones (Function): Un bloque de código reutilizable que puede ser llamado.
Ejemplo Símbolo (Symbol): Identificador único utilizado para propiedades de ojetos.
Ejemplo BigInt:Para manejar números enteros grandes.
