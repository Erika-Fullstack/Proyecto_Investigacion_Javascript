7. Operadores Especiales

Operador
Descripción
Ejemplo
Ternario
Simplifica condicionales
          let edad = 20;
          let tipo = (edad >= 18) ? 'Adulto' : 'Menor';
Typeof
Devuelve el tipo de una variable
          console.log(typeof 42);
          console.log(typeof 'Hola');
          console.log(typeof true);
instanceof
Verifica si un objeto es una instancia de una clase
          let fecha = new Date();
          console.log(fecha instanceof Date);
delete
ELimina una propiedad de un objeto
        let persona = { nombre: 'Ana', edad: 30 };
        delete persona.edad;
in
Verifica si una propiedad existe en un objeto
        let coche = { marca: 'Toyota', modelo: 'Corolla' };
        console.log('marca' in coche);
        console.log('color' in coche);
